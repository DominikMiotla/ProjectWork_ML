# PT_Iris: Albero Decisionale con Pivot sull'Iris Dataset

Questo progetto implementa una variante personalizzata dell'albero decisionale, denominata **Pivot Tree**, applicata al classico dataset Iris. Il modello utilizza strategie basate su pivot per la suddivisione dei dati, con l'obiettivo di migliorare interpretabilità e performance.

## Contenuto del Progetto

- `PivotTree.py` – Implementazione della struttura Pivot Tree.
- `RuleTree.py` – Estrazione di regole interpretabili dall'albero.
- `Utilis.py` – Funzioni di utilità (metriche, operazioni di supporto).
- `model_iris.py` – Script principale per allenare e valutare il modello.
- `pivot_tree_iris.pkl` – Modello salvato per inferenza o visualizzazione.
- `info_fit.txt` – Informazioni sulla fase di addestramento.
- `albero_con_pivot.txt` – Rappresentazione testuale dell’albero generato.
- `valutazione_modello.txt` – Risultati e metriche di valutazione del modello.
